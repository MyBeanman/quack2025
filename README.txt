This file was used to preform motor testing for the KRAKEN x60 motors. 

This was made pre-competition and utilizes 2024 game rules ext. This is not compatible with 2025 competition systems

For conversion, copy and paste code into a newer 2025 system with all 2025 dependencies installed. 

To install first initialize a VENV python version >= 12.0 run "py -m pip install robotpy" then run "py -m robotpy sync", respond no to updating the version.
